Dasar-dasar pemrograman Python

Materi pemrograman python ini disusun secara sederhana untuk pemula yang ingin belajar tentang pemrograman python. Terdiri dari 10 bab yang berisi tentang dasar-dasar pemrograman python. Materi beisikan antara lain:
Bab 	Tema 	Materi
Bab I 	Sejarah Python 	Materi 1
Bab II 	Instalasi Python 	Materi 2
Bab III 	Struktur data, Variable 	Materi 3
Bab IV 	Operasi Dasar 	Materi 4
Bab V 	Perulangan 	Materi 5
Bab VI 	If Condition 	Materi 6
Bab VII 	List 	Materi 7
Bab VIII 	Tuple 	Materi 8
Bab IX 	Dictionary 	Materi 9
Bab X 	Fungsi 	Materi 10

Tema tersebut diatas dapat dimanfaatkan untuk dasar dalam perkuliahan metode numerik, Komputasi Geofisika maupun di seismologi.
